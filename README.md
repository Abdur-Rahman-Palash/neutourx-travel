# NEUTOURX - Premium Travel Agency Platform

A modern, enterprise-grade travel booking platform built with Next.js 14, TypeScript, and premium UI/UX design. Inspired by leading travel sites like Trip.com, Booking.com, and Airbnb.

## Features

- **Premium Design**: Deep navy, luxury gradients, glassmorphism effects
- **Advanced Animations**: Framer Motion and GSAP for smooth, cinematic interactions
- **Responsive Layout**: Sticky header, expandable sidebar, main content areas
- **Modern Tech Stack**: Next.js App Router, TypeScript, Tailwind CSS
- **Performance Optimized**: Fast loading, smooth scrolling, optimized images

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion, GSAP
- **Icons**: Lucide React, React Icons
- **State Management**: Zustand
- **UI Components**: Custom components with premium styling
- **Additional**: Swiper.js, Lenis Smooth Scroll, React Query

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) to view the application.
